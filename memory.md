# project：build up a website 


## job description:
- we are a team from Ruan Lab in LSI, Zhejiang University, China.We have collected data of zebrafish embryos development. We need to build a website to present the data and make it accessible to the public. We desire to record all related information as detailed as possible, including single cell data, browser results, and gene regulatory networks. The website should be easy to navigate and provide clear explanations of data.
